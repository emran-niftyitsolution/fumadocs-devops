# Day 43: Container Security

## 🎯 What You'll Learn

- Container security risks
- Image scanning
- Runtime security

## 🐳 Container Security Risks

- Vulnerable base images
- Insecure configurations
- Exposed secrets

## 🛠️ Image Scanning

- Use tools like Trivy, Clair, or Anchore
- Scan images in CI/CD pipelines

## 🛡️ Runtime Security

- Use read-only file systems
- Drop unnecessary privileges
- Monitor running containers

## 📝 Hands-on Exercise

- Scan a Docker image with Trivy

## ❓ Assessment

- Name a container security risk
- How can you secure a running container?
