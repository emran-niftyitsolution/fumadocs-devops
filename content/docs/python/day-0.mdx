# Day 0: Introduction to Python

## 🐍 What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It's widely used in web development, data science, automation, and more.

## 🚀 Python's Philosophy

- **Readability counts:** Code should be easy to read and understand
- **Simple is better than complex:** Keep things simple
- **Explicit is better than implicit:** Be clear about what you're doing

## 📝 Your First Python Program

```python
print("Hello, World!")
```

## 🛠️ Setting Up Python

1. Download Python from python.org
2. Install it on your system
3. Verify installation: `python --version`
