# Day 46: Cloud Security Basics

## 🎯 What You'll Learn

- Shared responsibility model
- Cloud security risks
- Best practices for AWS, Azure, GCP

## ☁️ Shared Responsibility Model

Cloud providers secure the infrastructure, you secure your data, apps, and access.

## ⚠️ Cloud Security Risks

- Misconfigured storage (open S3 buckets)
- Overly permissive IAM roles
- Exposed secrets in code

## ✅ Best Practices

- Use least privilege for IAM
- Enable MFA for all accounts
- Encrypt data at rest and in transit
- Regularly audit resources

## 📝 Hands-on Exercise

- Review IAM roles in a cloud account (use a sandbox or free tier)
- Enable MFA for a user

## ❓ Assessment

- What is the shared responsibility model?
- Name a common cloud security risk.
