# Day 59: Kubernetes Volumes & Storage

## 🎯 What You'll Learn

- Volumes in Kubernetes
- PersistentVolume (PV) and PersistentVolumeClaim (PVC)
- Storage classes

## 💾 Volumes

- Allow data to persist beyond pod lifecycle
- Types: emptyDir, hostPath, configMap, secret, persistentVolumeClaim

## 🗄️ PersistentVolume & PersistentVolumeClaim

- **PV:** Cluster resource for storage
- **PVC:** Request for storage by a user

## 🏷️ Storage Classes

- Define types of storage (fast, standard, etc.)

## 🛠️ Example PVC YAML

```yaml
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: mypvc
spec:
  accessModes:
    - ReadWriteOnce
  resources:
    requests:
      storage: 1Gi
```
