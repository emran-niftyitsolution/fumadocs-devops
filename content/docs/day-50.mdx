# Day 50: Multi-Cloud & Hybrid Cloud

## 🎯 What You'll Learn

- What is multi-cloud? What is hybrid cloud?
- Benefits and challenges
- Design considerations

## 🌥️ What is Multi-Cloud?

Using more than one cloud provider (AWS + GCP, etc.)

## 🏢 What is Hybrid Cloud?

Mix of on-premises and cloud resources

## ⚖️ Benefits & Challenges

- Avoid vendor lock-in
- Complexity in networking, security, management

## 📝 Design Considerations

- Unified monitoring/logging
- Consistent IAM
- Data portability

## 📝 Hands-on Exercise

- Diagram a hybrid cloud architecture
- List pros and cons of multi-cloud
