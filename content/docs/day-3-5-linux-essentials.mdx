## ðŸ”§ File Operations & Permissions

### **Basic File Commands**

```bash
# List files and directories
ls -la

# Create directory
mkdir -p /opt/myapp

# Copy files
cp source.txt destination.txt
cp -r source_dir/ destination_dir/

# Move/rename files
mv oldname.txt newname.txt

# Remove files
rm file.txt
rm -rf directory/

# View file content
cat file.txt
less file.txt
head -10 file.txt
tail -10 file.txt
```

### **File Permissions**

```bash
# Understanding permissions
ls -la file.txt
# -rw-r--r-- 1 user group 1024 Jan 1 12:00 file.txt
#  ^ ^ ^
#  | | â””â”€â”€ Others (read)
#  | â””â”€â”€â”€â”€ Group (read)
#  â””â”€â”€â”€â”€â”€â”€ Owner (read/write)

# Change permissions
chmod 755 script.sh
chmod +x script.sh
chmod u+rw,g+r,o+r file.txt

# Change ownership
chown user:group file.txt
chown -R user:group directory/
```

### **Permission Examples**

```bash
# Common permission patterns
chmod 644 file.txt    # Owner: read/write, Group: read, Others: read
chmod 755 script.sh   # Owner: read/write/execute, Group: read/execute, Others: read/execute
chmod 600 private.txt # Owner: read/write, Group: none, Others: none
chmod 777 temp.txt    # Everyone: read/write/execute (not recommended)
```

## ðŸŒ SSH Basics

### **SSH Configuration**

```bash
# SSH to remote server
ssh user@server.com
ssh -p 2222 user@server.com

# SSH with key authentication
ssh -i ~/.ssh/id_rsa user@server.com

# SSH configuration file
cat ~/.ssh/config
```

### **SSH Key Management**

```bash
# Generate SSH key
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

# Copy public key to server
ssh-copy-id user@server.com

# Or manually copy
cat ~/.ssh/id_rsa.pub | ssh user@server.com "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys"
```

### **SSH Security**

```bash
# SSH server configuration
sudo nano /etc/ssh/sshd_config

# Security settings
PermitRootLogin no
PasswordAuthentication no
PubkeyAuthentication yes
Port 2222  # Change default port
```

## ðŸ”§ System Administration

### **Service Management**


### **Log Management**


### **User Management**

```bash
# Delete user
sudo userdel -r username

# List users
cat /etc/passwd
getent passwd
```
