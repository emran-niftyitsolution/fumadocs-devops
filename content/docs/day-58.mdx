# Day 58: Kubernetes ConfigMaps & Secrets

## 🎯 What You'll Learn

- What are ConfigMaps and Secrets?
- How to use them in pods
- Best practices

## 🗂️ ConfigMaps

- Store non-sensitive configuration data
- Mounted as environment variables or files

## 🔐 Secrets

- Store sensitive data (passwords, tokens)
- Base64-encoded, not encrypted by default

## 🛠️ Example Secret YAML

```yaml
apiVersion: v1
kind: Secret
metadata:
  name: mysecret
  namespace: default
type: Opaque
data:
  password: cGFzc3dvcmQ=
```
