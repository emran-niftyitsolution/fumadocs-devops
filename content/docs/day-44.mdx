# Day 44: Infrastructure as Code Security

## 🎯 What You'll Learn

- IaC security risks
- Tools for IaC scanning
- Best practices

## 🏗️ IaC Security Risks

- Hardcoded secrets
- Open security groups
- Unrestricted IAM roles

## 🛠️ Tools

- **Checkov:** Scans Terraform, CloudFormation
- **tfsec:** Terraform security
- **cfn-lint:** CloudFormation

## ✅ Best Practices

- Scan IaC in CI/CD
- Use least privilege
- Review changes before applying

## 📝 Hands-on Exercise

- Scan a Terraform file with Checkov or tfsec

## ❓ Assessment

- Name an IaC security risk
- What is the purpose of Checkov?
