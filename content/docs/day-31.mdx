# Day 31: Grafana for Visualization

## 🎯 What You'll Learn

- Setting up Grafana
- Connecting data sources (Prometheus, Loki)
- Building dashboards
- Alerting in Grafana

## 🖥️ Setting Up Grafana

- Download and install from [grafana.com](https://grafana.com/get)
- Start the server and access the web UI

## 🔗 Connecting Data Sources

- Add Prometheus as a data source
- Add Loki for logs

## 📊 Building Dashboards

- Create panels for CPU, memory, error rate
- Use variables for dynamic dashboards

## 🚨 Alerting in Grafana

- Set up alert rules on panels
- Integrate with Slack, email, PagerDuty

## 📝 Hands-on Exercise

- Build a dashboard showing CPU and memory usage
- Add an alert for high error rate

## ❓ Assessment

- How do you add a new data source in Grafana?
- What is a panel in Grafana?
