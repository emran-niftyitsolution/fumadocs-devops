# Day 62: Kubernetes Helm

## 🎯 What You'll Learn

- What is Helm?
- Helm charts
- Installing and managing apps with Helm

## ⛵ What is Helm?

Helm is a package manager for Kubernetes. It simplifies deployment and management of applications.

## 📦 Helm Charts

- Pre-configured application templates
- Can be customized with values

## 🛠️ Basic Helm Commands

```bash
helm repo add bitnami https://charts.bitnami.com/bitnami
helm install my-release bitnami/nginx
helm list
```
