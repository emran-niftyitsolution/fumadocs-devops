## üåê Advanced Networking

### Network Configuration

**Network Interfaces:**

```bash
# View network interfaces
ip addr show
ifconfig

# Show interface statistics
ip -s link show
ethtool eth0

# Configure IP address
sudo ip addr add 192.168.1.100/24 dev eth0
sudo ip link set eth0 up

# Remove IP address
sudo ip addr del 192.168.1.100/24 dev eth0
```

**Network Configuration Files:**

```bash
# Ubuntu/Debian
/etc/network/interfaces

# Example configuration:
auto eth0
iface eth0 inet static
    address 192.168.1.100
    netmask 255.255.255.0
    gateway 192.168.1.1
    dns-nameservers 8.8.8.8 8.8.4.4

# RHEL/CentOS
/etc/sysconfig/network-scripts/ifcfg-eth0

# Example configuration:
DEVICE=eth0
BOOTPROTO=static
IPADDR=192.168.1.100
NETMASK=255.255.255.0
GATEWAY=192.168.1.1
DNS1=8.8.8.8
DNS2=8.8.4.4
ONBOOT=yes
```

### Network Troubleshooting

**Connectivity Testing:**


```bash

# Show ARP table

# Show network connections

# Show interface statistics
cat /proc/net/dev
```

### Network Services


**DNS Configuration:**

```bash

sudo nano /etc/resolv.conf

# Example:

search example.com

# Make it persistent (Ubuntu)
sudo nano /etc/systemd/resolved.conf
```

**Firewall Management:**

```bash
# UFW (Ubuntu)
sudo ufw enable
sudo ufw allow 22/tcp
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw deny 23/tcp
sudo ufw status

# iptables
sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT
sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT
sudo iptables -A INPUT -p tcp --dport 443 -j ACCEPT
sudo iptables -A INPUT -j DROP

# Save iptables rules
sudo iptables-save > /etc/iptables/rules.v4
```

## üíæ Storage Management

### Disk Partitioning

**fdisk - Disk Partitioning:**

```bash
# List disk partitions
sudo fdisk -l

# Create new partition table
sudo fdisk /dev/sdb

# fdisk commands:
# n: New partition
# p: Print partition table
# d: Delete partition
# w: Write changes
# q: Quit without saving

# Example session:
# n (new partition)
# p (primary)
# 1 (partition number)
# Enter (default start)
# +10G (size)
# w (write and quit)
```

**parted - Advanced Partitioning:**

```bash
# Start parted
sudo parted /dev/sdb

# Create partition table
mklabel gpt

# Create partition
mkpart primary ext4 0% 100%

# Print partition table
print

# Quit
quit
```

### Filesystem Management

**Creating Filesystems:**

```bash
# Format partition with ext4
sudo mkfs.ext4 /dev/sdb1

# Format with XFS
sudo mkfs.xfs /dev/sdb1

# Format with Btrfs
sudo mkfs.btrfs /dev/sdb1

# Check filesystem
sudo fsck /dev/sdb1
```

**Mounting Filesystems:**

```bash
# Create mount point
sudo mkdir /mnt/data

# Mount filesystem
sudo mount /dev/sdb1 /mnt/data

# Mount with options
sudo mount -o defaults,noatime /dev/sdb1 /mnt/data

# Unmount
sudo umount /mnt/data

# Auto-mount on boot
echo "/dev/sdb1 /mnt/data ext4 defaults,noatime 0 2" | sudo tee -a /etc/fstab
```

### LVM (Logical Volume Management)

**LVM Concepts:**

```bash
# Physical Volume (PV)
# Volume Group (VG)
# Logical Volume (LV)

# Create physical volume
sudo pvcreate /dev/sdb1

# Create volume group
sudo vgcreate datavg /dev/sdb1

# Create logical volume
sudo lvcreate -L 10G -n datalv datavg

# Format logical volume
sudo mkfs.ext4 /dev/datavg/datalv

# Mount logical volume
sudo mount /dev/datavg/datalv /mnt/data
```

**LVM Management:**

```bash
# Extend logical volume
sudo lvextend -L +5G /dev/datavg/datalv
sudo resize2fs /dev/datavg/datalv

# Reduce logical volume
sudo lvreduce -L 8G /dev/datavg/datalv
sudo resize2fs /dev/datavg/datalv

# Add physical volume to volume group
sudo vgextend datavg /dev/sdc1

# Show LVM information
sudo pvs
sudo vgs
sudo lvs
```

### RAID Management

**Software RAID (mdadm):**

```bash
# Create RAID 1 (mirror)
sudo mdadm --create /dev/md0 --level=1 --raid-devices=2 /dev/sdb1 /dev/sdc1

# Create RAID 5
sudo mdadm --create /dev/md0 --level=5 --raid-devices=3 /dev/sdb1 /dev/sdc1 /dev/sdd1

# Format RAID device
sudo mkfs.ext4 /dev/md0

# Mount RAID device
sudo mount /dev/md0 /mnt/raid

# Check RAID status
sudo mdadm --detail /dev/md0
cat /proc/mdstat
```

**RAID Monitoring:**

```bash
# Monitor RAID health
sudo mdadm --monitor --daemonise --scan

# Add spare disk
sudo mdadm --add /dev/md0 /dev/sde1

# Remove failed disk
sudo mdadm --remove /dev/md0 /dev/sdb1
```

### Kernel Tuning

**sysctl Configuration:**

```bash
# View kernel parameters
sysctl -a

# Set kernel parameter
sudo sysctl -w vm.swappiness=10

# Make changes permanent
echo "vm.swappiness=10" | sudo tee -a /etc/sysctl.conf
sudo sysctl -p
```

**Common Tuning Parameters:**

```bash
# Memory management
vm.swappiness=10
vm.dirty_ratio=15
vm.dirty_background_ratio=5

# Network tuning
net.core.rmem_max=16777216
net.core.wmem_max=16777216
net.ipv4.tcp_rmem=4096 87380 16777216
net.ipv4.tcp_wmem=4096 65536 16777216

# File system tuning
fs.file-max=65536
fs.inotify.max_user_watches=524288
```

### Application Tuning

**Web Server Tuning:**

```bash
# Nginx tuning
sudo nano /etc/nginx/nginx.conf

# Worker processes

    worker_connections 1024;
    use epoll;

    sendfile on;
    tcp_nopush on;
    tcp_nodelay on;
```

**ulimit Configuration:**

```bash

# Set limits for current session

ulimit -u 2048

# Permanent limits
sudo nano /etc/security/limits.conf

# Example:

* hard nofile 65536
* soft nproc 2048
* hard nproc 2048
```

**cgroups Configuration:**

```bash

# Set memory limit
echo 100M > /sys/fs/cgroup/memory/myapp/memory.limit_in_bytes

# Add process to cgroup
echo $PID > /sys/fs/cgroup/memory/myapp/tasks
```

## üìä Assessment Questions

**Day 5 Quiz:**
