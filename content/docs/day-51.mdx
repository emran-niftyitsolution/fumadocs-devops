# Day 51: Cloud Automation

## 🎯 What You'll Learn

- Infrastructure as Code (IaC) for cloud
- Automation tools (Terraform, CloudFormation)
- CI/CD for cloud resources

## 🏗️ IaC for Cloud

- Define infrastructure in code
- Version control, repeatability

## 🛠️ Automation Tools

- **Terraform:** Multi-cloud IaC
- **CloudFormation:** AWS-native IaC
- **Pulumi:** Code-based IaC

## 🔄 CI/CD for Cloud

- Automate deployments and updates
- Use pipelines for testing and promotion

## 📝 Hands-on Exercise

- Write a Terraform or CloudFormation file for a simple resource

## ❓ Assessment

- What is IaC?
- Name a cloud automation tool.
