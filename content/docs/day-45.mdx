# Day 45: Compliance & Auditing

## 🎯 What You'll Learn

- What is compliance?
- Common frameworks (SOC2, PCI-DSS, HIPAA)
- Auditing practices

## 📝 What is Compliance?

Adhering to laws, regulations, and standards (e.g., GDPR, HIPAA).

## 🏛️ Common Frameworks

- **SOC2:** Service Organization Control
- **PCI-DSS:** Payment Card Industry Data Security Standard
- **HIPAA:** Health Insurance Portability and Accountability Act

## 🕵️ Auditing Practices

- Log all access and changes
- Regular reviews
- Automated compliance checks

## 📝 Hands-on Exercise

- Identify a compliance requirement for your project
- Set up a basic audit log (even if just a text file)

## ❓ Assessment

- What is the purpose of auditing?
- Name a compliance framework.
