# Day 49: Cloud Cost Optimization

## 🎯 What You'll Learn

- Why cost optimization matters
- Common sources of waste
- Tools and strategies

## 💸 Why Optimize Costs?

Cloud costs can spiral without monitoring and controls.

## 🛠️ Common Sources of Waste

- Unused resources (idle VMs, unattached storage)
- Over-provisioned instances
- Lack of auto-scaling

## ✅ Tools & Strategies

- Use cost explorer tools (AWS Cost Explorer, GCP Billing)
- Set budgets and alerts
- Rightsize resources
- Schedule non-prod resources to shut down

## 📝 Hands-on Exercise

- Review your cloud bill (or a sample)
- Identify one area to save costs

## ❓ Assessment

- Name a tool for cloud cost management
- What is rightsizing?
