## 📖 What is JavaScript?

JavaScript is a high-level, interpreted programming language that powers the interactive behavior of web pages. It runs in all modern browsers and is essential for web development.

## 🕰️ Brief History

- Created by Brendan Eich in 1995
- Originally called Mocha, then LiveScript, then JavaScript
- Standardized as ECMAScript (ES)

## 🌐 How JavaScript Runs

- JavaScript code is executed by the browser's JavaScript engine (e.g., V8 in Chrome)
- Runs on the client-side (in the user's browser)

## 📝 Hands-on Exercise

1. Open your browser's developer console (F12 or right-click → Inspect → Console)
2. Type:
   ```js
   console.log("Hello, JavaScript!");
   ```
3. Press Enter and see the output
