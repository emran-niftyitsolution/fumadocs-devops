# Day 41: Static Code Analysis

## 🎯 What You'll Learn

- What is static code analysis?
- Popular tools (SonarQube, ESLint, Bandit)
- Integrating static analysis into CI/CD

## 🧑‍💻 What is Static Code Analysis?

Analyzes code without running it. Finds bugs, style issues, and vulnerabilities.

## 🛠️ Tools

- **SonarQube:** Multi-language analysis
- **ESLint:** JavaScript/TypeScript
- **Bandit:** Python

## 🔗 Integrating into CI/CD

- Add as a pipeline step
- Fail builds on critical issues

## 📝 Hands-on Exercise

- Run ESLint or Bandit on a sample project
