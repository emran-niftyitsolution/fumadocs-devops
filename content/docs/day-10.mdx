### What is Cloud Computing?

**Cloud Computing Definition:**

```bash
# Cloud Computing is the delivery of computing services over the internet
# including servers, storage, databases, networking, software, analytics, and intelligence

# Key Characteristics:
# - On-demand self-service
# - Broad network access
# - Resource pooling
# - Rapid elasticity
# - Measured service
```

**Cloud Computing Benefits:**

```bash
# Cost Benefits:
# - Pay-as-you-go pricing
# - No upfront infrastructure costs
# - Reduced operational expenses
# - Economies of scale

# Performance Benefits:
# - Global infrastructure
# - Low latency
# - High availability
# - Auto-scaling

# Security Benefits:
# - Advanced security features
# - Compliance certifications
# - Regular security updates
# - DDoS protection

# Scalability Benefits:
# - Auto-scaling
# - Load balancing
# - Elastic resources
# - Global distribution
```

### Cloud Computing Models

**Service Models:**

```bash
# Infrastructure as a Service (IaaS)
# - Virtual machines
# - Storage
# - Networking
# - Operating systems
# - Examples: AWS EC2, Azure VMs, Google Compute Engine

# Platform as a Service (PaaS)
# - Development platforms
# - Application hosting
# - Database services
# - Middleware
# - Examples: AWS Elastic Beanstalk, Azure App Service, Google App Engine

# Software as a Service (SaaS)
# - Complete applications
# - Web-based access
# - Automatic updates
# - Multi-tenant
# - Examples: Salesforce, Office 365, Google Workspace
```

**Deployment Models:**

```bash
# Public Cloud
# - Third-party provider
# - Shared infrastructure
# - Pay-per-use
# - Examples: AWS, Azure, GCP

# Private Cloud
# - Dedicated infrastructure
# - On-premises or hosted
# - Full control
# - Examples: OpenStack, VMware vCloud

# Hybrid Cloud
# - Combination of public and private
# - Data and application portability
# - Flexibility
# - Examples: AWS Outposts, Azure Stack

# Multi-Cloud
# - Multiple cloud providers
# - Vendor lock-in avoidance
# - Best-of-breed services
# - Examples: AWS + Azure + GCP
```

## ðŸ” Cloud Security

### Identity and Access Management (IAM)

**AWS IAM:**

```bash
# Create IAM user
aws iam create-user --user-name myuser

# Create access key
aws iam create-access-key --user-name myuser

# Create IAM policy
aws iam create-policy \
    --policy-name my-policy \
    --policy-document file://policy.json

# Attach policy to user
aws iam attach-user-policy \
    --user-name myuser \
    --policy-arn arn:aws:iam::123456789012:policy/my-policy

# Create IAM role
aws iam create-role \
    --role-name my-role \
    --assume-role-policy-document file://trust-policy.json
```

**Azure AD:**

```bash
# Create service principal
az ad sp create-for-rbac --name myapp

# Assign role to service principal
az role assignment create \
    --assignee "app-id" \
    --role "Contributor" \
    --resource-group myResourceGroup

# Create managed identity
az identity create --name myidentity --resource-group myResourceGroup
```

**GCP IAM:**

```bash
# Create service account
gcloud iam service-accounts create my-service-account

# Grant role to service account
gcloud projects add-iam-policy-binding my-project \
    --member="serviceAccount:my-service-account@my-project.iam.gserviceaccount.com" \
    --role="roles/storage.admin"

# Create service account key
gcloud iam service-accounts keys create key.json \
    --iam-account=my-service-account@my-project.iam.gserviceaccount.com
```

### Cost Management Strategies

**Resource Optimization:**

```bash
# Right-sizing instances:
# - Monitor CPU, memory, and network usage
# - Use auto-scaling groups
# - Implement spot instances for non-critical workloads
# - Use reserved instances for predictable workloads

# Storage Optimization:
# - Use appropriate storage classes
# - Implement lifecycle policies
# - Compress data before storage
# - Use CDN for static content

# Network Optimization:
# - Use private IP addresses
# - Implement data transfer optimization
# - Use regional resources
# - Monitor bandwidth usage
```

**Cost Monitoring Tools:**

```bash
# AWS Cost Explorer
aws ce get-cost-and-usage \
    --time-period Start=2024-01-01,End=2024-01-31 \
    --granularity MONTHLY \
    --metrics BlendedCost \
    --group-by Type=DIMENSION,Key=SERVICE

# Azure Cost Management
az consumption usage list \
    --start-date 2024-01-01 \
    --end-date 2024-01-31

# GCP Billing
gcloud billing accounts list
gcloud billing projects describe my-project-id
```

**Cost Optimization Scripts:**

```bash
# AWS Cost Optimization Script
cat > aws_cost_optimization.sh << 'EOF'
#!/bin/bash

echo "=== AWS Cost Optimization Report ==="

# List unused EBS volumes
echo "Unused EBS Volumes:"
aws ec2 describe-volumes --filters Name=status,Values=available --query 'Volumes[*].[VolumeId,Size,VolumeType]' --output table

# List idle EC2 instances
echo "Idle EC2 Instances:"
aws ec2 describe-instances --filters "Name=instance-state-name,Values=running" --query 'Reservations[*].Instances[*].[InstanceId,InstanceType,State.Name]' --output table

# List unused security groups
echo "Unused Security Groups:"
aws ec2 describe-security-groups --query 'SecurityGroups[?length(Attachments)==`0`].[GroupId,GroupName]' --output table

# List S3 buckets with lifecycle policies
echo "S3 Buckets without Lifecycle Policies:"
aws s3api list-buckets --query 'Buckets[?not_null(Tags[?Key==`lifecycle`])].[Name]' --output table
EOF

chmod +x aws_cost_optimization.sh
```
