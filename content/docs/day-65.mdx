# Day 65: Kubernetes CI/CD

## 🎯 What You'll Learn

- CI/CD for Kubernetes
- Tools (ArgoCD, Flux, Jenkins X)
- GitOps principles

## 🔄 CI/CD for Kubernetes

- Automate build, test, deploy
- Use manifests or Helm charts

## 🛠️ Tools

- **ArgoCD:** GitOps continuous delivery
- **Flux:** GitOps for Kubernetes
- **Jenkins X:** CI/CD for Kubernetes

## 🚀 GitOps Principles

- Declarative configuration
- Version control as the source of truth
- Automated reconciliation

## 📝 Hands-on Exercise

- Set up ArgoCD or Flux in a cluster
- Deploy an app using GitOps

## ❓ Assessment

- What is GitOps?
- Name a CI/CD tool for Kubernetes.
