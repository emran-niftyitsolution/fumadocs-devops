# Day 36: SRE Tools & Automation

## 🎯 What You'll Learn

- Popular SRE tools (Terraform, Ansible, Chef, Puppet)
- Automation for reliability
- Integrating SRE tools into CI/CD

## 🛠️ SRE Tooling

- **Terraform:** Infrastructure as Code (IaC)
- **Ansible/Chef/Puppet:** Configuration management
- **Prometheus/Grafana:** Monitoring
- **PagerDuty/Opsgenie:** Incident response

## 🤖 Automation Strategies

- Automate repetitive tasks (backups, scaling, deployments)
- Use scripts, pipelines, and configuration management

## 📝 Hands-on Exercise

- Write a simple Ansible playbook to install Nginx
- Create a Terraform file to provision a VM (use a sandbox or try HashiCorp's playground)

## ❓ Assessment

- Name two SRE tools and their primary use
- Why is automation important for reliability?
