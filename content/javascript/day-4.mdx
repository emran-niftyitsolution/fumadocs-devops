# Day 4: Control Flow - if, else, switch

## 🎯 What You'll Learn

- `if`, `else if`, `else` statements
- `switch` statements
- Truthy and falsy values

## 📝 if, else if, else

```js
let score = 85;
if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else {
  console.log("C or below");
}
```

## 🔄 switch Statement

```js
let fruit = "apple";
switch (fruit) {
  case "apple":
    console.log("🍎");
    break;
  case "banana":
    console.log("🍌");
    break;
  default:
    console.log("Unknown fruit");
}
```

## ⚖️ Truthy and Falsy

- Falsy: `false`, `0`, `''`, `null`, `undefined`, `NaN`
- Everything else is truthy

## 📝 Hands-on Exercise

- Write an `if-else` to check if a number is positive, negative, or zero
- Try a `switch` statement for days of the week

## ❓ Assessment

- What is a falsy value?
- When would you use a `switch` statement?
