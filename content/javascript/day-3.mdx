# Day 3: Operators and Expressions

## 🎯 What You'll Learn

- Arithmetic operators
- Comparison operators
- Logical operators
- Assignment operators

## ➕ Arithmetic Operators

- `+`, `-`, `*`, `/`, `%`, `**`

```js
let sum = 5 + 3; // 8
let product = 4 * 2; // 8
let power = 2 ** 3; // 8
```

## 🔍 Comparison Operators

- `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`

```js
console.log(5 == "5"); // true
console.log(5 === "5"); // false
```

## ⚡ Logical Operators

- `&&` (and), `||` (or), `!` (not)

```js
let isAdult = age >= 18 && age < 65;
```

## 📝 Assignment Operators

- `=`, `+=`, `-=`, `*=`, `/=`, `%=`

```js
let x = 10;
x += 5; // x = 15
```

## 📝 Hands-on Exercise

- Try arithmetic and logical operations in the console
- Compare values using `==` and `===`

## ❓ Assessment

- What is the difference between `==` and `===`?
- Name two logical operators.
